# CSS Review: IsometricOffice Component
# File: clawsuite/src/components/agent-swarm/isometric-office.tsx
# Date: 2026-02-11

## Overview
The component renders a pixel-art virtual office for AI agents with a checkered floor,
desks, decorations, and animated agent avatars. Uses Tailwind CSS + inline styles.

---

## 5 Visual Improvement Suggestions

### 1. Add Drop Shadows to Furniture Elements for Depth
**Current:** Desks and furniture are flat with no shadows.
**Issue:** Lacks depth perception; elements float without grounding.
**Suggestion:**
```tsx
// Desk element - add shadow-lg and subtle transform
<div className="flex flex-col items-center drop-shadow-[0_4px_6px_rgba(0,0,0,0.4)]">
  <div className="h-8 w-10 rounded-t-sm border border-blue-500/40 bg-slate-800/60 shadow-inner">
```
Also add `shadow-xl` to the meeting table for prominence.

---

### 2. Implement Gradient Floor Instead of Flat Tiles
**Current:** Flat checkered floor with `#1a1a2e` / `#16213e` solid fills.
**Issue:** Floor looks static and lacks the polished "game" feel.
**Suggestion:** Add subtle radial gradient overlay for spotlight effect:
```tsx
// After CheckeredFloor SVG
<div className="absolute inset-0 bg-[radial-gradient(ellipse_at_center,rgba(59,130,246,0.08)_0%,transparent_70%)]" />
```
This creates a center spotlight that draws attention to active agents.

---

### 3. Enhance Agent Status Badges with Glow Effects
**Current:** Small 1.5px status dots with basic colors.
**Issue:** Status is hard to read at a glance; doesn't pop visually.
**Suggestion:**
```tsx
<div className={cn(
  'size-2 rounded-full ring-2',
  session.swarmStatus === 'running' && 'bg-blue-400 ring-blue-400/30 shadow-[0_0_8px_rgba(96,165,250,0.6)]',
  session.swarmStatus === 'thinking' && 'bg-amber-400 ring-amber-400/30 shadow-[0_0_8px_rgba(251,191,36,0.6)]',
  session.swarmStatus === 'complete' && 'bg-emerald-400 ring-emerald-400/30',
  ...
)} />
```
Ring + glow creates a radar-like pulse effect.

---

### 4. Add Ambient Particle Animation (Floating Dust/Sparkles)
**Current:** Only agent thinking bubbles animate; background is static.
**Issue:** Office feels "frozen" when no agents are active.
**Suggestion:** Add subtle floating particles component:
```tsx
function AmbientParticles() {
  return (
    <div className="pointer-events-none absolute inset-0 overflow-hidden">
      {Array.from({ length: 12 }).map((_, i) => (
        <motion.div
          key={i}
          className="absolute size-1 rounded-full bg-slate-400/20"
          style={{ left: `${10 + i * 8}%`, top: `${20 + (i % 3) * 25}%` }}
          animate={{ y: [-10, 10], opacity: [0.2, 0.5, 0.2] }}
          transition={{ duration: 3 + i * 0.5, repeat: Infinity, ease: "easeInOut" }}
        />
      ))}
    </div>
  )
}
```
Creates subtle life without distracting from agents.

---

### 5. Improve Typography Hierarchy with Text Shadows
**Current:** Agent names use `text-[10px] font-bold` with persona colors; role uses `text-[8px] text-slate-500`.
**Issue:** Labels can be hard to read against varying backgrounds.
**Suggestion:**
```tsx
// Agent name - add text shadow for legibility
<span className={cn(
  'mt-0.5 text-[10px] font-bold drop-shadow-[0_1px_2px_rgba(0,0,0,0.8)]',
  persona.color
)}>
  {persona.name}
</span>

// Role label - slightly larger with better contrast
<span className="text-[9px] text-slate-400 drop-shadow-sm font-medium">
  {persona.role}
</span>
```
Text shadows ensure readability on any background.

## Priority Ranking
1. Shadow/depth on furniture (highest impact)
2. Status glow effects (improves UX)
3. Text shadows (accessibility)
4. Floor gradient (atmosphere)
5. Ambient particles (polish)
