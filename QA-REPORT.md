## Bugs Found

### 1. Sidebar Overlay Blocks Interaction
**Severity:** HIGH  
**Description:** When sidebar is open, an overlay (`<button aria-label="Collapse navigation sidebar">`) covers the entire screen and intercepts pointer events, preventing clicks on any page elements.

**Repro Steps:**
1. Open ClawSuite at localhost:3000
2. Navigate to any page (e.g., /tasks)
3. Sidebar is open by default
4. Try clicking "New Task" button → fails, overlay intercepts

**Fix:** Add `pointer-events: none` to overlay when sidebar is collapsed, or fix z-index stacking.

---

### 2. Joyride Tour Intercepts Clicks
**Severity:** MEDIUM  
**Description:** React Joyride onboarding tour overlay blocks interaction with page elements.

**Repro Steps:**
1. Load page with fresh localStorage (first visit)
2. Joyride tour modal appears
3. Try clicking any button → tour overlay intercepts

**Fix:** Close joyride programmatically after first run, or add "Skip" button that's always accessible.

---

### 3. Navigation Links Not Detected
**Severity:** LOW (Test script issue)  
**Description:** Test couldn't find sidebar links because sidebar was in wrong state.

**Repro Steps:**
1. Run automated test
2. Sidebar needs to be in correct open/collapsed state

**Fix:** Test needs better wait states or manual verification.
